#+Author: ikyp

** Intro

  This repo is a little /program/ about ~status bar~ on *Window Manager*.

  Called *lstatus*

  It's very simple & clear.

** Usage

#+BEGIN_SRC 
make
make install
#+END_SRC

Be installed to ~/usr/local/bin~

Then, add ~lstatus &~ to *.xprofile* or *.xinitrc*
